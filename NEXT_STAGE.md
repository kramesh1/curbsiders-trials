# Next Stage

The repository is ready for attributable clinical review; it is not clinically
approved yet and is not currently reviewed or endorsed by The Curbsiders.

Use [REVIEW_HANDOFF.md](REVIEW_HANDOFF.md) for current counts, review-state
definitions, the first review packet, the trial-screening pilot, commands, and
known curation debt. Use [CURATION_GUIDE.md](CURATION_GUIDE.md) for record-level
review criteria.

The governing rule is simple: `pending` and `auto_triaged` model output remains
withheld. Only human-reviewed records carrying `review_status: "approved"` and
`reviewed_by` can enter publication sidecars.
